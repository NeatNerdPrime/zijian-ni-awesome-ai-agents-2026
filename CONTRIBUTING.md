# Contributing to Awesome AI Agents 2026

Thank you for your interest in contributing! 🎉

## How to Contribute

### Adding a New Project

1. Fork this repository
2. Add your project to the appropriate category in `README.md`
3. Follow the format: `- [Project Name](link) - Short description. ![GitHub stars](badge)`
4. Submit a Pull Request

### Guidelines

- **Real projects only** — The project must exist and be publicly accessible.
- **Relevant to AI agents** — The project should be directly related to AI agents, agent frameworks, tools, or resources.
- **Quality over quantity** — We prioritize well-maintained, actively developed projects.
- **Accurate descriptions** — Keep descriptions concise (one sentence) and factual.
- **Correct links** — Ensure all links point to the official repository or website.
- **Alphabetical order** — Within each category, entries should be in alphabetical order (where practical).

### Format

For GitHub projects:
```
- [Project Name](https://github.com/owner/repo) - Short description. ![GitHub stars](https://img.shields.io/github/stars/owner/repo?style=flat-square)
```

For non-GitHub projects:
```
- [Project Name](https://official-website.com/) - Short description.
```

### What We Don't Accept

- Projects that don't exist or are vaporware
- Duplicate entries
- Self-promotional spam without substance
- Projects unrelated to AI agents
- Broken links

## Suggesting a New Category

If you think a new category is needed, please open an issue first to discuss it.

## Reporting Issues

- Broken links
- Incorrect descriptions
- Outdated information
- Projects that no longer exist

Please open an issue with details.

## Code of Conduct

Be respectful, constructive, and collaborative. We're all here to build something awesome together.
